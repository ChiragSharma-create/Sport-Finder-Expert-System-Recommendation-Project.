This project is a rule-based expert system built in C++ that recommends a sport based on user preferences. The goal of this program was to simulate how an expert system makes decisions by taking user input and applying logical rules to generate a recommendation.

The program asks the user a series of structured questions about their preferences, such as whether they prefer team or solo activities, indoor or outdoor environments, endurance or quick bursts of energy, and whether they enjoy physical contact. Each input is validated to ensure the system only processes correct and meaningful data. Based on these responses, the program uses a set of conditional rules to determine the most suitable sport.

What makes this project interesting to me is how it models decision-making in a simplified AI system. Instead of using randomness or hardcoded outputs, I designed the program to follow a rule-based structure where different combinations of inputs lead to different outcomes. This approach reflects the core idea behind expert systems, where knowledge is represented through rules and used to simulate human-like reasoning.

A key part of this project was making sure the system is reliable and user-friendly. I focused on input validation, clear prompts, and consistent program flow so that the user experience feels smooth and logical. I also structured the code into separate functions for greeting, input handling, and decision-making, which made the program easier to debug and extend.

Through this project, I developed:

A strong understanding of rule-based systems and basic AI concepts
Experience designing decision logic using conditional structures
Skills in input validation and user interaction handling
Practice writing clean, modular C++ code

This project reflects my interest in how simple rule-based systems can be used to simulate intelligent decision-making. It also serves as a foundation for exploring more advanced AI techniques in the future, such as decision trees, machine learning models, or data-driven recommendation systems.
